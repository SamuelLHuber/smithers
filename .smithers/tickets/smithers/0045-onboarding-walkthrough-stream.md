# Produce a ~1-hour onboarding walkthrough/stream on a simple new project

Filed from user feedback on X (onboarding replies), 2026-06-17.

## Problem

*"An hour stream of you using it for something simple (ideally a new project) to
make onboarding a little easier. It's pretty configurable, so it'd be
interesting to see how *you* use it."* Users want to see the maintainer actually
use smithers end-to-end on a fresh project.

## Requirement

- Record (stream + keep VOD) a ~1-hour walkthrough starting from an empty/new
  project: `init` → hello-world → a real small task, showing the maintainer's
  actual workflow and configuration choices.
- Link it from get-started / the front page once published.

## Acceptance

- A published walkthrough video exists and is linked from onboarding.

> Content/devrel task (not code) — track to completion and link the artifact.
