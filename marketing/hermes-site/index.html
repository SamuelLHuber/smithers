Your agent's skills, faster, cheaper, more reliable.

Smithers lets your Hermes agent take on bigger jobs and actually get them done. It splits a job into steps, runs them at the same time so it's faster, uses a cheap model for the easy parts so it costs less, redoes only the bits that flake so it's reliable, and checks with you before anything risky. Free, open, runs right alongside Hermes.

A big task on its own

Slow, pricey, and easy to break

  • It tries to do the whole thing in one go
  • One model does every part, so it's slow and expensive
  • One step trips and you get half an answer, or nothing
  • If it gets interrupted, it starts over from scratch
The same task with Smithers

Faster, cheaper, and it finishes

  • The job is split into steps that run side by side
  • Cheap model for the easy parts, smart one only where it counts
  • A step that flakes is retried, so you get the full result
  • If it's interrupted, it picks up where it left off

What it does for your agent

Hermes already learns skills and does tasks for you. Smithers makes those skills do more, for less, without falling over.

⚡

Faster

It does several steps at the same time instead of one after another. A job that used to take all night can finish in a fraction of the time.

$

Cheaper

It uses a cheap, fast model for the simple steps and a smart one only where it really matters, and never pays to redo work it already finished.

✓

More reliable

Big jobs actually finish. If a step trips, it tries again instead of dropping everything, so you get the whole result, not half.

✦

Takes on more

Hand it bigger, multi-step jobs: research something all week, keep an eye on things, pull a report together. Not just one answer at a time.

⏸

Checks with you first

Before it deletes, sends, or spends anything, it asks you right in your chat, on Telegram, Discord, or Slack. No nasty surprises.

↩

Reports back

Kick a job off and walk away. When it's done, the result lands back in the same chat you started it from.

The boring magic underneath

Just like you can order an Uber, put your phone in your pocket, and trust the car will still show up, Smithers gives your agent's jobs that same promise: once one starts, you can trust it'll run all the way to the finish. And if something goes wrong on the way, your agent can restart it, retry it, or fix it.
