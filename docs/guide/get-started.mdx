<Steps>

<Step title="Run one command">

From inside your project:

```bash
bunx smithers-orchestrator init
```

</Step>

<Step title="Run your first workflow">

```bash
bunx smithers-orchestrator workflow run hello
```

A durable run completes and you see the output. Its prompt lives in `.smithers/prompts/hello.mdx` (plain Markdown). Edit it, run it again, watch the output change.

</Step>

</Steps>
