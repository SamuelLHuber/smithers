---
title: TUI Removed
description: The old smithers tui command is no longer shipped; use the CLI, GUI, or gateway surfaces instead.
---

`smithers tui` is not available in the current CLI. It was removed in `0.20.2` when the OpenTUI dependency and `apps/cli/src/tui/` tree were dropped from the package.

Use these supported surfaces instead:

```bash
bunx smithers-orchestrator ps --watch
bunx smithers-orchestrator events <run-id> --watch
bunx smithers-orchestrator chat <run-id> --follow
bunx smithers-orchestrator inspect <run-id> --watch
bunx smithers-orchestrator node <node-id> --run-id <run-id> --watch
bunx smithers-orchestrator tree <run-id> --watch
bunx smithers-orchestrator gui .
```

## Replacements

| Need | Use |
| --- | --- |
| Active run list | [`ps --watch`](/cli/overview) |
| Lifecycle event stream | [`events --watch`](/cli/overview) or [`logs --follow`](/cli/overview) |
| Agent transcript | [`chat --follow`](/cli/overview) |
| Run detail | [`inspect --watch`](/cli/overview) |
| Node attempts, tools, and output | [`node --watch`](/cli/overview), [`output`](/cli/overview), [`diff`](/cli/overview) |
| Rendered DevTools tree | [`tree --watch`](/cli/overview) |
| Desktop control plane | [`gui`](/cli/overview) and the [installation guide](/installation#desktop-gui) |
| Remote or browser control plane | [`up --serve`](/integrations/serve), [Gateway](/integrations/gateway), and [MCP Server](/integrations/mcp-server) |

Historical TUI product/design material is kept under `docs/design-prompts/` for reference, but it does not describe a shipped command.
