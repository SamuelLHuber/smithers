---
title: getRun
description: Gateway RPC for reading one run.
maturity: stable
apiVersion: v1
---

- Method: `getRun`
- Scope: `run:read`
- Transports: HTTP RPC, WebSocket RPC
- Request: `{ runId }`
- Response: `RunStateView`

Use this after a `GapResync` event (see `streamRunEvents`) to rebuild local state from scratch.
