---
title: getNodeDiff
description: Gateway RPC for node-level diff inspection.
maturity: stable
apiVersion: v1
---

- Method: `getNodeDiff`
- Scope: `run:read`
- Transports: HTTP RPC, WebSocket RPC
- Request: `{ runId, nodeId, iteration? }`
- Response: `GetNodeDiffRouteResult` (`payload` is `DiffBundle | GetNodeDiffStatPayload`)

Errors: `InvalidRequest`, `Unauthorized`, `Forbidden`, `RunNotFound`, `NodeNotFound`, `IterationNotFound`, `PayloadTooLarge`, `VcsError`, `Internal`.
