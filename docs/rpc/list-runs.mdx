---
title: listRuns
description: Gateway RPC for listing runs.
maturity: stable
apiVersion: v1
---

- Method: `listRuns`
- Scope: `run:read`
- Transports: HTTP RPC, WebSocket RPC
- Request: `{ filter?: { status?, limit? } }`
- Response: `RunSummary[]`

Errors are versioned as `v1` and include `InvalidRequest`, `Unauthorized`, `Forbidden`, and `Internal`.

Results are ordered newest first.
