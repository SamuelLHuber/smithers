---
title: getNodeOutput
description: Gateway RPC for task output inspection.
maturity: stable
apiVersion: v1
---

- Method: `getNodeOutput`
- Scope: `run:read`
- Transports: HTTP RPC, WebSocket RPC
- Request: `{ runId, nodeId, iteration? }`
- Response: `NodeOutputResponse`

Errors include `InvalidRequest`, `Unauthorized`, `Forbidden`, `RunNotFound`, `NodeNotFound`, `IterationNotFound`, `NodeHasNoOutput`, `PayloadTooLarge`, and `Internal`.
